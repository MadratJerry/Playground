# 重庆理工大学课程表导出

该脚本发布在 https://greasyfork.org/zh-CN/scripts/39366-%E9%87%8D%E5%BA%86%E7%90%86%E5%B7%A5%E5%A4%A7%E5%AD%A6%E8%AF%BE%E7%A8%8B%E8%A1%A8%E5%AF%BC%E5%87%BA

## 如何使用

登录教务系统之后，进入个人课表页面，会出现“导出”按钮。

直接点击下载即可，脚本自动识别当前学期和该学期第一周。
